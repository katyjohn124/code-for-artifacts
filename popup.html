使用教程 Usage Guide

本程序支持以下代码预览： • HTML+CSS+JS 组合代码：直接点击 "Preview Code" 按钮 • React 组件（App.js + App.css）：依次点击 "Collect CSS" 和 "Collect JSX"，预览窗口将自动打开 • Vue 单文件组件：直接点击 "Preview Code" 按钮
This extension supports preview for: • HTML+CSS+JS combination: Click "Preview Code" directly • React components (App.js + App.css): Click "Collect CSS" then "Collect JSX", preview window will open automatically • Vue single-file components: Click "Preview Code" directly
